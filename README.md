### 시나리오 선정 이유
``
e커머스와 관련된 핵심 기능을 구현하면서 실무에서 고민될 동시성 이슈 처리, 성능 최적화, 단위 테스트 등을 경험해 보기 위해 이 시나리오를 선정했습니다.

``
